# 🌸 Gulabi High School Website

A modern, responsive school website built using **Next.js**, **React**, and **Tailwind CSS**. This project showcases essential school information with smooth animations and a clean UI.

---

## 🚀 Overview

This website includes the following pages:

- 🏠 Home
- 📖 About
- 🎓 Admission
- 📞 Contact
- 🖼️ Gallery

---

## 🛠️ Tech Stack

- **Next.js** (App Router)
- **React**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**

---

## 📁 Project Structure

```bash

src/
│
├── app/
│   ├── page.tsx              # Home page
│   ├── about/page.tsx
│   ├── admission/page.tsx
│   ├── contact/page.tsx
│   ├── gallery/page.tsx
│   └── globals.css
│
├── components/
│   ├── layout/
│   │   ├── Navbar.tsx
│   │   └── Footer.tsx
│   │
│   └── ui/
│       ├── PageTransition.tsx
│       └── SectionHeading.tsx

````

## ⚙️ Getting Started

### 1️⃣ Install Dependencies

```bash
npm install
````

---

### 2️⃣ Run Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```

---

## 🏗️ Build for Production

```bash
npm run build
```

---

## ▶️ Start Production Server

```bash
npm run start
```

## 📌 Notes

* Uses the **Next.js App Router (`app/` directory)**.
* Fully responsive design.
* Smooth animations powered by **Framer Motion**.
* Easily deployable on:

  * **Vercel**
  * **Netlify**

---

## 🌐 Deployment

To deploy quickly:

